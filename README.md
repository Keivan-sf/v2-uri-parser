# v2-uri-parser
V2ray URI parser for xray core

Currently supports: `vless`

```
V2ray URI parser

Usage: v2parser [OPTIONS] <URI>

Arguments:
  <URI>

Options:
      --socksport <socksport>
      --httpport <httpport>
  -h, --help                   Print help
  -V, --version                Print version
```
